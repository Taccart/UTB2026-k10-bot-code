## GC2145 specific:
`GC2145` is the camera model of Unihiker K10 board.
Camera can output YUV/YCbCr422, RAW Bayer and RGB565 and provice images up to 1600x1200.

* technical documentation : [GC2145 CSP DataSheet release V1.0_20131201](GC2145%20CSP%20DataSheet%20release%20V1.0_20131201.pdf)
* Low level code fro linux [codebrowser website's gc2145.html ](https://codebrowser.dev/linux/linux/drivers/media/i2c/gc2145.c.html) copied in [code_browser_gc2145.md](code_browser_gc2145.md)
